### Semantic HTML

Semantic HTML communicates meaning and structure to browsers and assistive tech.

- **Key point** -> Use elements like `<nav>`, `<main>`, `<article>`.
- **Key point** -> Better SEO and accessibility.
- **Gotcha** -> Excessive `<div>` soup hides structure.

Example:
```html
<main>
  <article>
    <h1>Title</h1>
  </article>
</main>
```
