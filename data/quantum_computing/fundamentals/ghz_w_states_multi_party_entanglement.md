### GHZ and W States — Multi-party Entanglement

**What it is:**
Entanglement of 3+ parties has inequivalent classes: states can't always be converted to each other by local operations and classical communication (SLOCC). GHZ and W are the two distinct SLOCC classes for 3 qubits. They have strikingly different robustness and correlation structure.

**GHZ state (n qubits):**
```
|GHZ_n⟩ = (|0⟩^⊗n + |1⟩^⊗n) / √2
|GHZ₃⟩  = (|000⟩ + |111⟩) / √2
```
Generalization of the Bell |Φ⁺⟩ to n parties.

**W state (n qubits):**
```
|W_n⟩ = (1/√n) Σᵢ |0...1ᵢ...0⟩       (exactly one qubit is in |1⟩)
|W₃⟩  = (|001⟩ + |010⟩ + |100⟩) / √3
```

**Creation circuits:**

GHZ₃ — single H + chain of CNOTs (depth O(log n) with a tree):
```python
from qiskit import QuantumCircuit
qc = QuantumCircuit(3)
qc.h(0); qc.cx(0, 1); qc.cx(0, 2)     # (|000⟩ + |111⟩)/√2
```

W₃ — requires non-Clifford parametrized gates:
```python
import numpy as np
qc = QuantumCircuit(3)
qc.ry(2*np.arccos(1/np.sqrt(3)), 0)    # put q0 into √(1/3)|0⟩+√(2/3)|1⟩
qc.ch(0, 1)                            # controlled-H
qc.cx(1, 2); qc.cx(0, 1); qc.x(0)
```

**Robustness — particle loss:**
| Property | GHZ | W |
|---|---|---|
| Lose 1 qubit (trace out) | becomes separable mixture ½(\|0…0⟩⟨0…0\| + \|1…1⟩⟨1…1\|) → NO residual entanglement | remains entangled (pairwise concurrence survives) |
| Max pairwise entanglement | zero (multipartite-only) | nonzero (2/n concurrence between any pair) |

This means **W is robust**: even if one party disappears, the remaining n−1 are still entangled. GHZ is fragile: lose one qubit, lose everything.

**SLOCC inequivalence:**
No combination of local invertible operations + classical communication can convert GHZ ↔ W. For 3 qubits there are exactly two genuinely tripartite SLOCC classes (plus biseparable and fully separable): GHZ-class and W-class.

**Correlations:**
GHZ has maximal n-party correlations: ⟨Z₁Z₂...Zₙ⟩ = 1, and measuring one Zᵢ determines all others. GHZ violates Mermin inequality maximally, giving an all-vs-nothing contradiction with local realism for n ≥ 3.

W has 2-party correlations (concurrence between each pair > 0) but weaker n-party correlations. Useful as a "one-excitation" model in coherent transport (BEC, spin chains).

**Use cases:**
- **GHZ**: quantum metrology (Heisenberg-limited phase estimation on n sensors gives Δφ ~ 1/n vs 1/√n for separable), secret sharing, quantum anonymous broadcasting, some QEC stabilizers.
- **W**: loss-tolerant entanglement distribution, leader election without symmetry breaking, quantum memory with graceful degradation.

**Dicke states (generalization of W):**
```
|D_n^k⟩ = (1/√C(n,k)) Σ permutations(|0⟩^(n−k) |1⟩^k)
|D_n^1⟩ = |W_n⟩
```
States of definite Hamming weight k, symmetric under qubit permutation. Generated by specific ladder circuits; appear in particle-conserving ansatz for VQE (chemistry, Hubbard model).

**Cluster / graph states:**
Yet another multipartite resource, different SLOCC class from GHZ or W. Each vertex ↔ qubit, each edge ↔ CZ. Universal for measurement-based quantum computation.

**Measuring multipartite entanglement:**
Bipartite entropy isn't enough beyond 2 qubits. Useful invariants:
- **Residual tangle τ** (3-tangle): nonzero only for GHZ-class.
- **Geometric measure**: distance from nearest product state.
- **Genuine-multipartite-entanglement (GME) witnesses** for experimental certification.

**Rule of thumb:** For maximal n-party correlations and metrology use GHZ; for robustness under particle loss use W; remember they are fundamentally different entanglement classes — no local protocol interconverts them.
