### Phase Gates (S, T): Clifford vs Non-Clifford

**What they are:** Diagonal single-qubit gates that add a relative phase to |1⟩. S is a π/2 phase; T is a π/4 phase (the "half" of S). S is in the Clifford group — T is not, and that distinction drives the cost of fault-tolerant quantum computing.

**Matrices:**
```
S = [[1, 0],          T = [[1, 0          ],
     [0, i]]               [0, e^{iπ/4}   ]]

S² = Z                T² = S       T⁸ = I
```

S adds phase e^{iπ/2} = i on |1⟩. T adds phase e^{iπ/4} = (1+i)/√2 on |1⟩.

**Relation to rotations (up to global phase):**
- S = Rz(π/2) · e^{iπ/4}
- T = Rz(π/4) · e^{iπ/8}

**The Clifford group Cₙ:** Normalizer of the Pauli group — unitaries U such that U·Pₙ·U† ⊆ Pₙ. Generated by {H, S, CNOT}. Preserves the Pauli basis under conjugation.

| Gate | Clifford? | Conjugation of X / Z         |
|------|-----------|------------------------------|
| H    | yes       | HXH=Z, HZH=X                 |
| S    | yes       | SXS†=Y, SZS†=Z               |
| CNOT | yes       | maps Paulis → Paulis         |
| T    | **no**    | TXT† is not a Pauli (it's (X+Y)/√2) |

**Why T is expensive (fault-tolerance):** In most stabilizer codes (surface code, color code), Clifford gates are implemented **transversally** — cheap and naturally fault-tolerant. T gates cannot be transversal in any error-detecting stabilizer code (Eastin-Knill theorem), so they require **magic state distillation**: consuming many noisy |T⟩ = T|+⟩ states to produce few clean ones. T-count and T-depth dominate fault-tolerant resource estimates.

**Universality:** Clifford alone is efficiently classically simulable (Gottesman-Knill). Adding T makes {H, S, CNOT, T} universal and unlocks quantum advantage.

**Qiskit:**
```python
from qiskit import QuantumCircuit
qc = QuantumCircuit(1)
qc.s(0)     # S gate, Rz(π/2) phase
qc.sdg(0)   # S† (S-dagger)
qc.t(0)     # T gate, Rz(π/4) phase
qc.tdg(0)   # T† (T-dagger)

# Equivalence
from qiskit.quantum_info import Operator
import numpy as np
from qiskit.circuit.library import SGate, TGate
assert np.allclose(Operator(SGate()).data, Operator(TGate()).data @ Operator(TGate()).data)
```

**T-count examples:** Toffoli ≈ 7 T, controlled-S ≈ 2 T, adders are T-count-dominated.

**Rule of thumb:** S is free, T is expensive — when you design a fault-tolerant circuit, the resource metric that matters most is the T-count (sometimes T-depth).
